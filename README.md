# 拿貨單管理系統 (Product List)

單頁 webapp，用於管理拿貨單與庫存。支援匯入/匯出 Excel（.xlsx），依名稱與顏色分組計算數量與總額。

## 使用方式

### 線上使用（GitHub Pages）

開啟：**https://kwtsang101016.github.io/productlist/**

### 本地使用

1. 下載或 clone 本 repo。
2. 用瀏覽器開啟 `index.html`（或直接雙擊檔案）。

## 需求

- Excel 檔案需包含工作表：**待結款**、**库存**
- 「待結款」工作表欄位：名称、颜色、价格、数量、发货日期、退货日期（可簡體/繁體）

## 功能

- 匯入 Excel、編輯資料、匯出 Excel
- 待結款：篩選、排序、清零、計算總和（數量總和與總額）
- 库存：剩餘庫存計算
